# حاسبة الجمارك

موقع ثابت بسيط لحساب التكلفة التقريبية بعد سعر المنتج والشحن والجمارك والرسوم الإضافية.

## الحالة الحالية

المشروع يعمل كملفات static فقط:

- `index.html` يحتوي الصفحة كاملة.
- `404.html` يحوّل الروابط الخاطئة إلى الصفحة الرئيسية.
- GitHub Actions ينشر الملفات إلى GitHub Pages بدون Node أو Vite أو React.

## النشر

النشر يتم من workflow:

```text
.github/workflows/deploy-pages.yml
```

الخطوات الحالية:

1. إنشاء مجلد `dist`.
2. نسخ `index.html` إلى `dist`.
3. نسخ `404.html` إلى `dist` إن وجد.
4. رفع `dist` إلى GitHub Pages.

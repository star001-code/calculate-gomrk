حول النظام

تم تحويل النظام إلى نسخة static حتى يعمل على GitHub Pages، مع الحفاظ على شكل النظام القديم والاعتماد على ملفات البيانات الموجودة داخل الكود.

الميزات التي تحتاج Backend مثل تسجيل الدخول الحقيقي، الجلسات، OCR الحقيقي، أو قاعدة بيانات حية لا تعمل على GitHub Pages. لكن البحث والحساب والتعرفة تعمل من ملفات static.
